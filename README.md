# Mangalam HDPE Pipes - Product Page

**🌐 Live Demo:** [View the live site on Netlify](YOUR_NETLIFY_LINK_HERE)

A fully responsive, interactive product landing page built strictly with vanilla web technologies. This project was developed to match specific Figma design requirements, focusing on pixel-perfect UI, performance, and clean code architecture without the use of any external frameworks or libraries.

## 🚀 Features

- **Smart Sticky Header:** A navigation bar that dynamically appears when scrolling up and hides when scrolling down to maximize screen real estate.
- **Interactive Image Gallery:** A custom-built product carousel featuring thumbnail navigation and a precise hover-to-zoom effect on the main image.
- **Dynamic Content Sections:** Includes an interactive FAQ accordion and a tabbed navigation system for the manufacturing process section.
- **Horizontal Scroll Carousels:** Smooth, button-controlled horizontal scrolling for the Applications and Testimonials sections.
- **Fully Responsive Design:** Fluid layouts utilizing modern CSS Grid and Flexbox that adapt beautifully across desktop, tablet, and mobile devices.

## 🛠️ Technologies Used

- **HTML5:** Semantic structuring and accessibility best practices.
- **CSS3:** Custom styling, Grid/Flexbox layouts, smooth transitions, and media queries for responsiveness.
- **JavaScript (Vanilla):** DOM manipulation, event handling, math calculations for image zooming, and bulletproof safety checks. 
- *No CSS frameworks (like Bootstrap/Tailwind) or JS libraries (like jQuery/React) were used.*
